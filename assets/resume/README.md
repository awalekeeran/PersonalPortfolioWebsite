# Resume/CV Folder

Place your resume PDF file here.

## File Name
`resume.pdf`

## Requirements

- **Format**: PDF (recommended for universal compatibility)
- **File Size**: Keep under 2MB for fast downloads
- **Content**: Should include:
  - Contact information
  - Professional summary
  - Work experience
  - Education
  - Skills
  - Certifications (if applicable)
  - Projects/Portfolio highlights

## Alternative Formats

You can also include other formats:
- `resume.docx` - Microsoft Word format
- `resume-web.html` - HTML version for web viewing

## Tips for Creating a Great Resume

1. **Keep it Concise**: 1-2 pages maximum
2. **Use Professional Formatting**: Clean, readable layout
3. **Tailor for Your Audience**: Highlight relevant experience
4. **Proofread**: No typos or grammatical errors
5. **Update Regularly**: Keep information current

## Tools for Creating Resumes

- [Canva](https://www.canva.com/create/resumes/) - Free templates
- [Resume.io](https://resume.io/) - Professional resume builder
- [Overleaf](https://www.overleaf.com/) - LaTeX resume templates
- Microsoft Word - Built-in templates
- Google Docs - Free templates

## Download Link in Portfolio

The download link is set up in `index.html`:

```html
<a href="assets/resume/resume.pdf" class="btn btn-primary" download>Download CV</a>
```

Make sure your `resume.pdf` file is placed in this folder for the download link to work.
